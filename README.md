# JVO Auto-Serviços — Oficina Mecânica

Este é o site oficial da **JVO Auto-Serviços**, uma oficina mecânica multimarcas localizada em Umuarama-PR, com mais de 25 anos de experiência no mercado automotivo. O site foi desenvolvido para proporcionar uma experiência moderna, rápida e informativa para os clientes.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema React:

- **[TanStack Start](https://tanstack.com/router/v1/docs/guide/start/overview)**: Framework Full-stack para React com roteamento tipado e SSR (Server-Side Rendering).
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Engine de estilização de alto desempenho.
- **[shadcn/ui](https://ui.shadcn.com/)**: Componentes de UI acessíveis e reutilizáveis.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para animações fluidas e interativas.
- **[Lucide React](https://lucide.dev/)**: Conjunto de ícones minimalistas.
- **[TypeScript](https://www.typescriptlang.org/)**: Tipagem estática para maior segurança e produtividade.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build e servidor de desenvolvimento ultra-rápido.
- **[Bun](https://bun.sh/)**: Runtime e gerenciador de pacotes rápido.

## 🚀 Como Rodar o Projeto

Siga os passos abaixo para configurar e executar o projeto localmente:

### Pré-requisitos

Certifique-se de ter o [Bun](https://bun.sh/) instalado em sua máquina.

### Instalação

1. Clone o repositório (ou acesse a pasta do projeto).
2. Instale as dependências:

```bash
bun install
```

### Execução

Para iniciar o servidor de desenvolvimento com hot-reload:

```bash
bun dev
```

O site estará disponível em [http://localhost:3000](http://localhost:3000) (ou na porta indicada no terminal).

### Build de Produção

Para gerar a versão otimizada para produção:

```bash
bun build
```

## 📁 Estrutura do Projeto

- `src/routes/`: Contém as páginas e rotas da aplicação (roteamento baseado em arquivos).
- `src/components/`: Componentes reutilizáveis divididos em `ui` (base) e `site` (específicos).
- `src/assets/`: Imagens, logos e recursos visuais.
- `src/lib/`: Funções utilitárias e configurações globais (ex: dados de contato).
- `src/styles.css`: Configurações globais do Tailwind CSS v4.
